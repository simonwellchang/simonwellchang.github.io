9. 數學公式

主題使用了 MathJax 開源庫來實現對數學公式的支持，使用 $$ 標記。

$$ \left\{ \begin{aligned} N & = pq \\ \varphi(n) & = (p-1)(q-1)\\ \end{aligned} \right. \Rightarrow N - \varphi(n) + 1 = p + q $$

10. JSFiddle

引入 JSFiddle 網站的代碼範例，在主題目錄 layouts/shortcodes 文件夾下的 jsfiddle.html 對該標籤進行定義。

11. Codepen

引入 Code Pen 網站的代碼演示，在主題目錄 layouts/shortcodes 文件夾下的 codepen.html 對該標籤進行定義。

See the Pen 🐍 Snake Rush by Ritchie Zhu (@laozhu) on CodePen.

12. 聲享 PPT

引入 聲享 PPT 演示文稿，在主題目錄 layouts/shortcodes 文件夾下的 shengxiang.html 對該標籤進行定義。

13. 本地視頻

主題使用了video.js 播放視頻文件，你還可以自己定義視頻的封面，在主題目錄layouts/shortcodes 文件夾下的video.html 對該標籤進行定義。

14. 網易云音樂

主題文章中可以輕鬆插入網易云音樂 的指定音樂，你可以根據需要將音樂設置為自動播放，在主題目錄layouts/shortcodes 文件夾下的music.html 對該標籤進行定義。

**注意：由於版權問題，網易已經禁止外站分享版權音樂，該 shortcode 已經無法正常使用。 **

15. Gist 代碼片段

除了本地的代碼片段，主題中可使用 GitHub 的 Gist 服務輕鬆插入代碼片段。
